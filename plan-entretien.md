# Plan d'entretien — Flotte Bus / Autocars

## 1. Catalogue des actions d'entretien

### A — Moteur & fluides

| Code | Action | Description |
|------|--------|-------------|
| A01 | Vidange moteur + filtre à huile | Remplacement huile moteur et filtre à huile |
| A02 | Filtre à carburant diesel | Remplacement filtre(s) à gasoil (pré-filtre + filtre principal) |
| A03 | Filtre à air moteur | Remplacement élément filtrant air moteur |
| A04 | Filtre de ventilation cabine | Remplacement filtre(s) habitacle / ventilation passagers |
| A05 | Filtre séparateur d'eau | Purge et/ou remplacement du séparateur eau-carburant |
| A06 | Contrôle niveau et appoint AdBlue | Vérification circuit AdBlue, appoint si nécessaire |
| A07 | Courroie(s) accessoires | Contrôle tension et état, remplacement si nécessaire |
| A08 | Courroie de distribution (si applicable) | Remplacement selon préconisation constructeur |
| A09 | Contrôle turbocompresseur | Inspection visuelle, jeu, étanchéité |

### B — Refroidissement

| Code | Action | Description |
|------|--------|-------------|
| B01 | Contrôle circuit de refroidissement | Niveau, état liquide, étanchéité durites |
| B02 | Remplacement liquide de refroidissement | Vidange et remplacement intégral |
| B03 | Contrôle thermostat | Vérification ouverture/fermeture correcte |
| B04 | Contrôle ventilateur / visco-coupleur | Fonctionnement, état des pales |

### C — Freinage

| Code | Action | Description |
|------|--------|-------------|
| C01 | Contrôle épaisseur plaquettes/garnitures | Mesure usure, remplacement si seuil atteint |
| C02 | Contrôle disques / tambours | Mesure épaisseur / ovalisation, remplacement si nécessaire |
| C03 | Réglage frein de parc | Réglage et contrôle efficacité |
| C04 | Remplacement liquide de frein | Purge et remplacement intégral |
| C05 | Contrôle circuit pneumatique de freinage | Étanchéité, pression, dessiccateur d'air |
| C06 | Remplacement cartouche dessiccateur | Remplacement élément filtrant du dessiccateur d'air |
| C07 | Contrôle ABS / EBS / ESP | Diagnostic électronique, capteurs |
| C08 | Contrôle ralentisseur (retarder) | Niveau huile, fonctionnement |

### D — Transmission

| Code | Action | Description |
|------|--------|-------------|
| D01 | Vidange boîte de vitesses automatique | Remplacement huile + filtre BVA |
| D02 | Vidange boîte de vitesses manuelle | Remplacement huile boîte manuelle |
| D03 | Vidange pont / différentiel | Remplacement huile de pont |
| D04 | Contrôle embrayage (si BVM) | Garde, usure disque, butée |
| D05 | Contrôle arbre de transmission / cardans | Jeu, croisillons, graissage |
| D06 | Graissage général châssis | Graissage de tous les points (pivots, articulations) |

### E — Direction & suspension

| Code | Action | Description |
|------|--------|-------------|
| E01 | Contrôle direction assistée | Niveau huile, étanchéité, jeu |
| E02 | Contrôle suspension pneumatique | État coussins, compresseur, capteurs hauteur |
| E03 | Contrôle amortisseurs | Fuite, efficacité |
| E04 | Contrôle rotules / biellettes | Jeu, état, remplacement si nécessaire |
| E05 | Géométrie / parallélisme | Réglage train avant et arrière |

### F — Pneumatiques

| Code | Action | Description |
|------|--------|-------------|
| F01 | Contrôle pression et usure pneus | Mesure profondeur sculpture, pression, usure anormale |
| F02 | Permutation / remplacement pneus | Rotation ou remplacement selon usure |
| F03 | Contrôle roues et fixations | Serrage écrous, état jantes |

### G — Électricité & électronique

| Code | Action | Description |
|------|--------|-------------|
| G01 | Contrôle batteries | Tension, état bornes, capacité |
| G02 | Contrôle éclairage complet | Phares, feux, clignotants, éclairage intérieur |
| G03 | Contrôle alternateur / démarreur | Charge, état courroie |
| G04 | Diagnostic électronique général | Lecture défauts calculateurs (moteur, BVA, ABS, portes…) |
| G05 | Contrôle essuie-glaces et lave-glace | État balais, fonctionnement gicleurs, niveau liquide |
| G06 | Mise à jour logicielle (si applicable) | MAJ firmware calculateurs selon bulletins constructeur |

### H — Carrosserie & sécurité

| Code | Action | Description |
|------|--------|-------------|
| H01 | Contrôle portes passagers | Mécanisme ouverture/fermeture, capteurs anti-pincement |
| H02 | Contrôle rampe PMR / agenouillement | Fonctionnement rampe, système kneeling |
| H03 | Contrôle issues de secours | Marteaux brise-vitre, trappes, signalétique |
| H04 | Contrôle extincteurs | Date validité, pression, accessibilité |
| H05 | Contrôle ceintures de sécurité | Fonctionnement enrouleur, état sangle |
| H06 | Contrôle sièges et fixations | Serrage, état, mécanisme rabattable |
| H07 | Contrôle soutes / coffres (autocars) | Mécanisme, vérins, serrures |
| H08 | Contrôle rétroviseurs et caméras | Réglage, état, fonctionnement caméra recul |

### J — Climatisation & chauffage

| Code | Action | Description |
|------|--------|-------------|
| J01 | Contrôle climatisation | Charge gaz, fonctionnement, étanchéité |
| J02 | Entretien climatisation (recharge) | Recharge gaz réfrigérant, contrôle compresseur |
| J03 | Contrôle chauffage auxiliaire (Webasto/Eberspächer) | Fonctionnement, bougie de préchauffage, pompe |
| J04 | Nettoyage / désinfection circuit aéraulique | Évaporateur, gaines, filtres |

### K — Spécifique GNV (Gaz Naturel)

| Code | Action | Description |
|------|--------|-------------|
| K01 | Contrôle réservoirs GNV | Inspection visuelle, conformité (requalification périodique) |
| K02 | Contrôle détendeur haute/basse pression | Étanchéité, pression de sortie |
| K03 | Contrôle soupapes de sécurité | Fonctionnement, date de validité |
| K04 | Contrôle capteur de fuite gaz | Calibration, fonctionnement alarme |
| K05 | Remplacement bougies d'allumage | Remplacement bougies moteur gaz |
| K06 | Contrôle vanne d'arrêt gaz | Fonctionnement ouverture/fermeture |
| K07 | Filtre à gaz | Remplacement élément filtrant circuit gaz |

## 3. Périodicité par catégorie de véhicule

### 3.1 Autocars (Diesel)

**Véhicules** : Mercedes Tourismo, Tourino, Setra 411, Setra 516, MAN Lion Coach R10/R08/R07, Neoplan, Scania Touring, Isuzu Turquoise, Temsa MD9/MD7/LD12

| Forfait | Périodicité | Remarques |
|---------|-------------|-----------|
| **F1** — Entretien courant | Tous les **30 000 km** ou **6 mois** | |
| **F2** — Entretien intermédiaire | Tous les **60 000 km** ou **12 mois** | Remplace F1 |
| **F3** — Grand entretien | Tous les **120 000 km** ou **24 mois** | Remplace F2 |
| **F4** — Transmission | Tous les **120 000 km** ou **3 ans** | En même temps que F3 |
| **F5** — Climatisation | **1x/an** (avant été) | |
| **F6** — Sécurité | **1x/an** + avant contrôle technique | Soutes incluses (H07) |
| **F8** — Pneumatiques | Selon usure, contrôle à chaque F1 | |

#### Spécificités par modèle

| Modèle | Particularité |
|--------|---------------|
| Mercedes Tourismo | BVA 8 rapports — F4 avec D01. Retarder intégré — C08 à chaque F2 |
| Mercedes Tourino | Midibus — volumes fluides réduits |
| Setra 411 / 516 | Plateforme Mercedes — mêmes intervalles. 516 HD : retarder ZF |
| MAN Lion Coach R10 | Moteur MAN D26 — vidange 40 000 km possible (consulter bulletin) |
| MAN Lion Coach R08/R07 | R08 : D20, R07 : D08. Intervalles identiques |
| Neoplan | Plateforme MAN — suivre intervalles MAN |
| Scania Touring | Moteur Scania DC09/DC13 — vidange possible à 45 000 km (huile longue durée) |
| Isuzu Turquoise | Moteur Cummins — vidange 20 000 km. F1 plus fréquent |

## 5. Cycle d'entretien type — Exemple autocar (120 000 km)

```
  0 km ─── F1 (vidange)
 30 000 ── F1
 60 000 ── F2 (intermédiaire)
 90 000 ── F1
120 000 ── F3 (grand entretien) + F4 (transmission)

Annuel : F5 (clim) + F6 (sécurité)
À l'usure : F8 (pneumatiques)
```

## 6. Cycle d'entretien type — Exemple bus urbain GNV (80 000 km)

```
  0 km ─── F1 (vidange)
 20 000 ── F1
 40 000 ── F2 (intermédiaire) + F7 (GNV contrôle)
 60 000 ── F1
 80 000 ── F3 (grand entretien) + F4 (transmission) + F7 (GNV complet)

Semestriel : F5 (clim) + F6 (sécurité)
Mensuel : Contrôle pneus
À l'usure : F8 (pneumatiques complet)
```
